The "God Mode" Personal Assistant for developers and data-driven teams.

VibeCode Agent leverages the OpenClaw engine to provide an elite, sandboxed AI assistant directly in your Telegram.

## Features
- **BYOB (Bring Your Own Bot):** Use your own private Telegram bot as the interface.
- **GitHub Auth:** Secure onboarding and repo-aware intelligence.
- **Sandbox Security:** Every execution happens in an isolated Docker container.
- **Data-Driven:** Deep integration with GSC, Google Analytics, and GitHub APIs.

## Project Structure
- `/scripts`: Automation for user provisioning and gateway management.
- `/plugins`: Custom VibeCode connectors for Google and GitHub.
- `/web`: The VibeCode dashboard for authentication and bot management.
